Como Participar

  1. Preencha seu nome no campo indicado.

  2. Clique no botão "Qual o meu prêmio?" para ver seu resultado.

  3. Compartilhe o resultado com o tatuador pelo WhatsApp ou Instagram.

  4. Resgate seu prêmio apresentando a mensagem compartilhada ao tatuador.

  5. Para validar o prêmio é necessário estar seguindo @kevii_ink.
Aqui você deve inserir o seu nome de usuário do Instagram.
